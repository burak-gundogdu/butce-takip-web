# Butce Takip Web

Kisisel butce ve yatirim takip uygulamasi.

## Kurulum

```bash
npm install
npm run dev
```

## Vercel'e Deploy

1. GitHub'a push et
2. vercel.com -> Import Project -> GitHub repo sec
3. Deploy!

## Ozellikler

- Firebase Auth ile kisisel hesap
- Firestore ile bulut veri saklama
- Gelir/Gider takibi
- Yatirim portfoyu (Hisse, Kripto, Altin, Gumus, Doviz)
- AI Asistan (Groq API - ucretsiz)
- Istatistik ve analiz
- Abonelik takibi
- Borc takibi
- CSV disa aktarimi
